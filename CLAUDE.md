# CLAUDE.md

## 基本的な振る舞い

- ファイル末尾には必ず改行を入れること
- `LocalStack` を除くライブラリ・コンテナイメージには、特に理由がない限り最新の安定版を使用すること
- 最新バージョン等の情報は必ずインターネットで調べてから使用すること
- 英語で考えて、チャットコメントやドキュメントは日本語で書くこと

## 設計

- パッケージ構成やアーキテクチャの設計方針は `docs/ARCHITECTURE.md` を参照すること

## GitHub

- GitHub情報（issue, PR, リポジトリ等）の取得・操作には `gh` コマンドを使用すること
